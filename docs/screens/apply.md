# 新規申し込み画面仕様書

## 基本情報

| 項目 | 内容 |
|------|------|
| 画面ID | SCR-APPLY |
| 画面名 | 新規サービス申し込み |
| ファイル | apply.html |
| URL | /apply.html |
| 認証 | 要ログイン |

## 画面概要

新しいサービスを申し込むためのステップ型フォーム。「サービス選択→確認→完了」の3ステップで構成される。

## ステップ構成

| ステップ | 名称 | 内容 |
|---------|------|------|
| ① | サービス選択 | 申し込むサービスとプランを選択する |
| ② | 確認 | 選択内容を確認する |
| ③ | 完了 | 申し込み完了メッセージを表示する |

## 操作仕様

### サービス選択変更

サービスを変更すると、プランのセレクトボックスが選択したサービスに対応するプラン一覧に自動更新される。

### 「確認画面へ」ボタン押下

選択されたサービス・プランの情報をステップ2の確認画面に反映し、ステップ2に遷移する。

### 「← 戻る」ボタン押下（ステップ2）

ステップ1のサービス選択画面に戻る。入力内容は保持される。

### 「申し込む」ボタン押下（ステップ2）

申し込み処理を実行し、ステップ3（完了画面）に遷移する。

### 「契約サービス一覧へ」ボタン押下（ステップ3）

契約サービス一覧画面（services.html）に遷移する。

## 画面遷移

| 遷移元 | 操作 | 遷移先 |
|--------|------|--------|
| サイドバー | 新規申し込みリンク | この画面（ステップ1） |
| ステップ1 | 確認画面へ | ステップ2 |
| ステップ2 | ← 戻る | ステップ1 |
| ステップ2 | 申し込む | ステップ3 |
| ステップ3 | 契約サービス一覧へ | 契約サービス一覧 |
