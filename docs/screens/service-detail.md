# サービス詳細画面仕様書

## 基本情報

| 項目 | 内容 |
|------|------|
| 画面ID | SCR-SERVICE-DETAIL |
| 画面名 | サービス詳細 |
| ファイル | service-detail.html |
| URL | /service-detail.html?id={サービスID} |
| 認証 | 要ログイン |

## 画面概要

選択されたサービスの契約情報を詳細表示する画面。利用中のサービスについては解約操作を行うことができる。

## URLパラメータ

| パラメータ | 必須 | 説明 |
|-----------|------|------|
| id | ○ | サービスID（例: SV001） |

## 表示項目

### 契約情報

| No. | 項目名 | 表示形式 | 説明 |
|-----|--------|----------|------|
| 1 | サービスID | テキスト | サービスの一意識別子 |
| 2 | サービス名 | テキスト | サービスの正式名称 |
| 3 | カテゴリ | テキスト | サービスカテゴリ |
| 4 | ステータス | バッジ | 利用中（緑）・申請中（黄）・解約済（赤） |
| 5 | プラン | テキスト | 契約プラン名 |
| 6 | 月額料金 | 通貨 | カンマ区切り + 「円」 |
| 7 | 契約日 | 日付 | YYYY/MM/DD形式 |
| 8 | オプション | テキスト | 契約中のオプション一覧（読点区切り）。なければ「なし」 |

### 操作ボタン

| No. | ボタン名 | 種類 | 表示条件 | 動作 |
|-----|----------|------|----------|------|
| 1 | ← 一覧に戻る | アウトラインボタン | 常に表示 | 契約サービス一覧に遷移 |
| 2 | このサービスを解約する | 危険ボタン（赤） | ステータスが「利用中」の場合のみ | 解約確認モーダルを表示 |

## 操作仕様

### 「← 一覧に戻る」ボタン押下

契約サービス一覧画面（services.html）に遷移する。

### 「このサービスを解約する」ボタン押下

解約確認モーダルを表示する。

### 解約確認モーダル

| 要素 | 内容 |
|------|------|
| タイトル | 「解約確認」 |
| メッセージ | 「このサービスを解約してよろしいですか？」 |
| 注意書き | 「※ 解約すると月末で利用が停止されます。再契約には新規申し込みが必要です。」（赤字） |
| キャンセルボタン | モーダルを閉じる。何もしない |
| 解約するボタン | 解約処理を実行し、完了メッセージを表示後、契約サービス一覧に遷移する |

## エラーハンドリング

| 条件 | 動作 |
|------|------|
| URLパラメータ「id」に該当するサービスが存在しない | 「サービスが見つかりません」とエラーメッセージを表示 |

## 画面遷移

| 遷移元 | 操作 | 遷移先 |
|--------|------|--------|
| 契約サービス一覧 | 詳細ボタン | この画面 |
| この画面 | 一覧に戻る | 契約サービス一覧 |
| この画面 | 解約実行 | 契約サービス一覧 |
